## 📌 Project Overview

RetailPulse is an AI-powered retail analytics platform developed to help retailers analyze sales performance, understand customer purchasing behavior, and improve business decisions using Machine Learning and Data Analytics.

The application provides an interactive dashboard built with Streamlit where users can upload retail datasets, visualize sales trends, segment customers, and generate business insights.

---

# 🎯 Objectives

- Analyze retail sales data
- Visualize business KPIs
- Perform customer segmentation using K-Means
- Forecast future demand
- Support inventory planning
- Provide an interactive analytics dashboard

# ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/RetailPulse.git
```

Go to project folder

```bash
cd RetailPulse
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
python -m streamlit run app.py
```
